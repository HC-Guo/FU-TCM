TCM_final 上传说明

如果要上传到 GitHub，先确认 docs/large_files.tsv。超过 100 MiB 的文件不能用普通 Git 直接提交。

只上传代码和小文件

git add README.md index.html .gitignore .env.example docs scripts benchmark identity_generation llamafactory_qwen35 *.py *.sh *.txt *.json
git commit -m "Organize TCM_final project files"
git push

需要上传大文件

git lfs install
git lfs track "*.json" "*.jsonl" "*.zip" "*.jpg" "*.png"
git add .gitattributes
git add .
git commit -m "Add TCM datasets with LFS"
git push

如果仓库没有 LFS 额度，建议把大压缩包放到 GitHub Release 或网盘，只在仓库里保留说明和小样本。
