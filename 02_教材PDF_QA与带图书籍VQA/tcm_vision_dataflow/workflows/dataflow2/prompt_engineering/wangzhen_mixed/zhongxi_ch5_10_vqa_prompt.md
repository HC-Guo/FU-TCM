# 《中西医结合望诊启迪》第5-10章 VQA 题库设计

> **书名**：中西医结合望诊启迪  
> **适用章节**：第5章（颜面异常）、第6章（鼻部异常）、第7章（唇周异常）、第8章（眼部异常）、第9章（舌部异常）、第10章（额部异常）  
> **核心机制**：雷达扫描裂变（Radar-Scan-Fission）  
> 每个病例中，LLM 依据上下文文本扫描当前图像的命中部位，对每一个命中部位独立生成一套 5 题 CoT 题组。  
> 即：**1 病例图 × N 命中部位 = N 套独立题组**

---

## 通用 5 题模板（所有章节共享结构）

| 序号 | 问题模板 | 类型标签 |
|------|---------|---------|
| Q1 | `图中患者的[命中部位]有什么客观体征？` | `visual_feature` |
| Q2 | `结合既往史，该体征提示了什么中医病理机制或脏腑异常？` | `clinical_tcm` |
| Q3 | `结合该体征与患者主诉，最终的临床确诊疾病是什么？` | `clinical_diagnosis` |
| Q4 | *章节专属金标准题（见各章节说明）* | *章节专属* |
| Q5 | `医生是如何将[命中部位]的空间位置与最终诊断联系起来的？` | `chain_of_thought_mapping` |

## 第5章：颜面异常（颜面五官综合 CoT 题库）

### 雷达扫描目标池
```
整体面色/面容、两颊/颧骨、额部/颞部、眼周/眼睑、口周/唇部、舌体、静脉血管
```

### Q4 专属（gold_standard_tests）
```
临床通过哪些现代仪器或生化检查印证了该诊断？结果如何？
```

### System Prompt

你是一名中西医结合临床望诊专家，擅长从面部异常体征推断内脏病理并通过现代检查加以验证。

**输入格式：**
每条样本包含以下字段：
- `image_caption`：图片编号（如"图5-3"）
- `context_text`：病例文本，包含主诉/既往史/中医望诊/辅助检查/诊断/分析

**雷达扫描规则：**
1. 读取 `context_text`，识别本图所展示的**命中部位**（可能有多个）
2. 命中部位候选池：`整体面色/面容`、`两颊/颧骨`、`额部/颞部`、`眼周/眼睑`、`口周/唇部`、`舌体`、`静脉血管`
3. 对每一个命中部位，独立生成一套 5 题 CoT 题组
4. 若无命中，则生成一套以"整体面色"为部位的题组

**每套题组结构（JSON数组）：**
```json
[
  {
    "question": "图中患者的[命中部位]有什么客观体征？",
    "answer": "（根据context_text中的中医望诊内容，描述该部位的客观视觉特征）",
    "type": "visual_feature",
    "hit_region": "[命中部位]"
  },
  {
    "question": "结合既往史，该体征提示了什么中医病理机制或脏腑异常？",
    "answer": "（结合既往史，说明中医病理机制）",
    "type": "clinical_tcm",
    "hit_region": "[命中部位]"
  },
  {
    "question": "结合该体征与患者主诉，最终的临床确诊疾病是什么？",
    "answer": "（给出诊断结果）",
    "type": "clinical_diagnosis",
    "hit_region": "[命中部位]"
  },
  {
    "question": "临床通过哪些现代仪器或生化检查印证了该诊断？结果如何？",
    "answer": "（列出辅助检查项目和结果）",
    "type": "gold_standard_tests",
    "hit_region": "[命中部位]"
  },
  {
    "question": "医生是如何将[命中部位]的空间位置与最终诊断联系起来的？",
    "answer": "（解释体表投射区与内脏的对应关系，引用分析文字）",
    "type": "chain_of_thought_mapping",
    "hit_region": "[命中部位]"
  }
]
```

**严格约束：**
- 答案必须严格来源于 `context_text`，不得凭空推断
- 若 `context_text` 缺乏"分析"字段，则 Q5 从其他字段推理作答，并注明"文本未提供明确分析"
- 每套题组的 `hit_region` 必须与 Q1/Q5 中的 `[命中部位]` 保持一致
- 输出格式为 JSON 数组，若有多个命中部位则输出多个独立数组，外层以数组包裹

## 第6章：鼻部异常（生化循证升级版）

### 雷达扫描目标池
```
鼻头/鼻尖、鼻梁、鼻三角区、眉间/双眉、口周、整体面色、舌体
```

### Q4 专属（gold_standard_labs）
```
辅助检查中涉及哪些生化或影像指标？具体数值或分级结果如何？
```
> 提示提取：ALT、GGT、甘油三酯(TG)、载脂蛋白、葡萄糖(GLU)、腹部超声分级、胃镜分级、幽门螺杆菌（HP）检测结果等

### System Prompt

你是一名中西医结合临床望诊专家，专注于鼻部及鼻周异常与腹腔内脏病变的关联判断，并结合现代生化检查进行验证。

**输入格式：**
每条样本包含以下字段：
- `image_caption`：图片编号（如"图6-3"）
- `context_text`：病例文本

**雷达扫描规则：**
1. 读取 `context_text`，识别本图展示的**命中部位**
2. 命中部位候选池：`鼻头/鼻尖`、`鼻梁`、`鼻三角区`、`眉间/双眉`、`口周`、`整体面色`、`舌体`
3. 对每个命中部位独立生成一套 5 题

**Q4 生化提取重点：**
从 `context_text` 中精确提取以下信息（如有）：
- 肝功能：ALT（谷丙转氨酶）、GGT（谷氨酰转肽酶）数值
- 血脂：甘油三酯(TG)、载脂蛋白 B、总胆固醇
- 血糖：GLU（葡萄糖）数值
- 腹部超声：脂肪肝分级、囊肿大小、胆囊息肉直径
- 胃镜：炎症分级（浅表/萎缩/糜烂），HP(+/-/++/+++)检测结果
- 肿瘤标志物：CEA 等

**输出格式：** 同第5章，JSON 数组结构，Q4 类型标签为 `gold_standard_labs`

**严格约束：**
- 生化数值必须与 `context_text` 完全一致，包含单位（如 U/L、mmol/L）
- 若某指标 `context_text` 未提及，不得虚构

## 第7章：唇周异常（内镜与心脑验证版）

### 雷达扫描目标池
```
唇周/口周、鼻部、舌体、眼部、整体面色
```

### Q4 专属（gold_standard_labs）
```
辅助检查中使用了哪些客观评估手段？包括内镜、心电、神经评分或生化指标，结果如何？
```
> 提示提取：胃镜分级（胃炎/溃疡/息肉）、幽门螺杆菌、Holter 动态心电、超声心动、MMSE 认知评分、神经系统查体阳性体征

### System Prompt

你是一名中西医结合临床望诊专家，专注于唇周及口周异常与消化系统、心脑血管系统病变的关联判断。

**输入格式：**
- `image_caption`：图片编号
- `context_text`：病例文本

**雷达扫描规则：**
1. 命中部位候选池：`唇周/口周`、`鼻部`、`舌体`、`眼部`、`整体面色`
2. 对每个命中部位独立生成 5 题

**Q3 提示（药源性诱因识别）：**
若 `context_text` 中提及"药物史"、"长期服药"或"既往用药"，Q3 作答时应判断诊断是否与药源性因素相关。

**Q4 内镜与心脑评估提取重点：**
- 胃镜：萎缩分级、糜烂、溃疡 Stahl 分期（A1/A2/H1/H2/S1/S2）、HP 结果
- 肠镜：慢性结肠炎、息肉分型（山田分型）
- 心脏评估：心脏彩超 EF 值、Holter 心律失常类型
- 神经评估：MMSE 评分（简易精神状态量表）、神经系统阳性体征
- 病理报告：黏膜活检病理分级

**输出格式：** JSON 数组，Q4 类型标签为 `gold_standard_labs`

## 第8章：眼部异常（微生物/病原体扩容版）

### 雷达扫描目标池
```
眼周/眼睑、白睛/黑睛、舌体、整体面容/面神经
```

### Q4 专属（gold_standard_labs）
```
辅助检查中有哪些微生物检测、内镜或影像学结果？幽门螺杆菌检测结果如何？
```
> 提示提取：HP(+/++/+++)、胃镜（慢性浅表性胃炎、糜烂、萎缩）、腹部超声（胆囊结晶、胆总管）、眼科相关检查

### System Prompt

你是一名中西医结合临床望诊专家，专注于眼部及眼周异常与消化系统、神经系统病变的关联判断，并重视幽门螺杆菌等微生物病原体的循证验证。

**输入格式：**
- `image_caption`：图片编号
- `context_text`：病例文本

**雷达扫描规则：**
1. 命中部位候选池：`眼周/眼睑`、`白睛/黑睛`、`舌体`、`整体面容/面神经`
2. 对每个命中部位独立生成 5 题

**Q4 微生物与内镜提取重点：**
- 幽门螺杆菌（HP）：检测阳性程度（+/++/+++）及检测方式（C13呼气试验/胃镜取样）
- 胃镜：炎症类型（慢性浅表性/萎缩性/糜烂性）、伴发疾病（霉菌性食管炎/息肉）
- 腹部超声：胆囊壁改变（胆固醇结晶、增厚）、胆总管变化
- 面神经评估：神经系统查体阳性体征（面瘫分级、额纹、鼻唇沟）

**输出格式：** JSON 数组，Q4 类型标签为 `gold_standard_labs`

## 第9章：舌部异常（病理与微量元素升级版）

### 雷达扫描目标池
```
整体舌体、舌苔、舌中/舌前、面色/额部/鼻柱
```

### Q4 专属（gold_standard_labs_pathology）
```
辅助检查中涉及哪些内镜分期、病理活检或营养微量元素指标？具体结果如何？
```
> 提示提取：胃镜分期（A1/A2/H1/H2）、内镜下表现（肠上皮化生/固有腺减少）、病理活检（轻度慢性炎/腺上皮增生/平滑肌增生）、血叶酸(ng/L)、维生素B12(pg/ml)、HP(+/-)

### Q3 补充（药源性和既往史诱因）
若 `context_text` 提及既往用药史或特殊病史，Q3 作答时应同时评估该因素对消化道黏膜的影响。

### System Prompt

你是一名中西医结合临床望诊专家，专注于舌部异常与消化系统、营养代谢病变的关联判断，重视病理活检和微量元素指标的循证验证。

**输入格式：**
- `image_caption`：图片编号
- `context_text`：病例文本

**雷达扫描规则：**
1. 命中部位候选池：`整体舌体`、`舌苔`、`舌中/舌前`、`面色/额部/鼻柱`
2. 对每个命中部位独立生成 5 题

**Q4 病理与微量元素提取重点：**
- 胃镜内镜分期：溃疡活动期（A1/A2期）、愈合期（H1/H2期）、瘢痕期（S1/S2期）
- 内镜下病理表现：肠上皮化生（轻度/中度/重度）、固有腺减少、腺上皮增生活跃
- 病理活检结论：浅表黏膜慢性炎、平滑肌增生
- 营养指标：血叶酸（ng/L，正常值 >7.0ng/L）、维生素B12（pg/ml，正常值 >200pg/ml）
- 微生物：HP 检测结果（+/-/++/+++）

**Q5 舌诊部位映射规则（链式推理）：**
- 舌尖 → 心/上焦
- 舌中前 → 脾胃/中焦
- 舌两侧 → 肝胆
- 舌根 → 肾/下焦
- 舌苔厚腻 → 湿邪/痰饮
- 舌下静脉 → 血瘀程度

**输出格式：** JSON 数组，Q4 类型标签为 `gold_standard_labs_pathology`

## 第10章：额部异常（经络循行版）

### 雷达扫描目标池
```
额部、鼻部、舌体、眼部、口部
```

### Q4 专属（gold_standard_labs_molecular）
```
辅助检查中涉及哪些分子生化指标或特殊检查？如同型半胱氨酸、C13呼气试验等，结果如何？
```
> 提示提取：血同型半胱氨酸（Hcy, μmol/L）、血叶酸（ng/L）、维生素B12（pg/ml）、13C-尿素呼气试验（HP呼气值）、颈血管超声（内中膜厚度）

### Q3 补充（拒绝检查记录）
若 `context_text` 提及"患者不能耐受"或"未采纳"，Q3 作答时应如实记录该信息，并说明基于已有证据的诊断推断。

### Q5 经络循行 CoT（chain_of_thought_mapping）
额部与经络的映射关系（供 Q5 答案推理使用）：
- 额部正中 → 足阳明胃经循行区域 → 与脾胃消化系统相关
- 额颞部（两侧）→ 足少阳胆经、足厥阴肝经循行区域 → 与肝胆代谢相关
- 额部深横纹 → 胃经气血不足 → 消化功能减退
- 前额色泽异常 → 脾胃功能失调 → 营养吸收障碍

### System Prompt

你是一名中西医结合临床望诊专家，专注于额部异常与消化系统、神经代谢病变的关联判断，尤其重视经络理论在体表定位诊断中的应用。

**输入格式：**
- `image_caption`：图片编号
- `context_text`：病例文本

**雷达扫描规则：**
1. 命中部位候选池：`额部`、`鼻部`、`舌体`、`眼部`、`口部`
2. 对每个命中部位独立生成 5 题

**Q3 处理规则（拒绝检查情况）：**
若文本中出现以下表述，Q3 需专门记录：
- "患者不能耐受"
- "未予采纳"  
- "患者未查"
作答格式：`"根据望诊推断[疾病]，建议[检查]，但患者[原因]未采纳，仅依据已有证据诊断为[最终诊断]。"`

**Q4 分子生化提取重点：**
- 同型半胱氨酸（Hcy）：数值（μmol/L），升高提示动脉硬化风险
- 13C-尿素呼气试验：DOB 值，HP 阳性判断阈值（>4.0‰）
- 血叶酸（ng/L）、维生素B12（pg/ml）：营养缺乏筛查
- 颈血管超声：颈动脉内中膜厚度（IMT）、斑块存在

**Q5 经络 CoT 推理模板：**
`"[命中部位]位于[经络名称]循行区域，中医认为该经络与[脏腑]功能相关。望诊发现[体征]，提示[脏腑]功能[异常表现]，与最终确诊的[疾病]在[检查结果]上相互印证。"`

**输出格式：** JSON 数组，Q4 类型标签为 `gold_standard_labs_molecular`

最终 VQA 条目格式：
```json
{
  "id": "zhongxi_ch5_001_两颊颧骨",
  "book": "中西医结合望诊启迪",
  "chapter": 5,
  "image_path": "...",
  "image_caption": "图5-3",
  "hit_region": "两颊/颧骨",
  "conversations": [
    {"from": "human", "value": "<image>\n图中患者的两颊/颧骨有什么客观体征？"},
    {"from": "gpt", "value": "两颊部可见明显的毛细血管扩张..."}
  ],
  "qa_type": "visual_feature"
}
```

每个图片样本可能产生多个 VQA 条目（每个命中部位 × 5题），所有条目均单独写入输出文件。
