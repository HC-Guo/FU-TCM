Meta Reasoning 辨证数据流程

从 meta_reasoning.json 生成结构化辨证数据，并对辨证参数进行复核。

流程

python scripts/convert/convert_bianzheng.py
python scripts/verify/verify_bianzheng.py

环境变量

export OPENAI_API_KEY="你的API_KEY"
export OPENAI_BASE_URL="https://cc.580ai.net/v1"
export META_REASONING_MODELS="claude-sonnet-4-6,deepseek-reasoner"

核心文件

路径说明
data/raw/meta_reasoning.json原始病案数据，共 799 条
data/processed/bianzheng_minimax.jsonl训练集转换结果，599 条
data/processed/bianzheng_minimax_test.jsonl测试集转换结果，200 条
configs/mapping_table.json辨证映射规则
scripts/convert/convert_bianzheng.py结构化转换脚本
scripts/verify/verify_bianzheng.py辨证参数复核脚本
